# Heap Sort

## Categories

- Quasilinear (log-linear) growth
- Unstable
- In-Place

## Time Complexity

- Worst: `O(n log n)`
- Best:
  - Distinct keys: `O(n log n)`
  - Equal keys: `O(n)`
- Average: `O(n log n)`

## Space Complexity

- Total: `O(n)`
- Auxiliary: `O(1)`
