# Bubble Sort

## Categories

- Polynomial (cuadratic) growth
- Comparison sort
- Stable
- In-Place

## Time Complexity

### Comparisons

- Worst: `O(n²)`
- Best: `O(n)`
- Average: `O(n²)`

### Swaps

- Worst: `O(n²)`
- Best: `O(1)`
- Average: `O(n²)`

## Space Complexity

- Total: `O(n)`
- Auxiliary: `O(1)`
